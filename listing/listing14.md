### Что выведет данная программа и почему?
```
func main() {
  slice := []string{"a", "a"}

  func(slice []string) {
    slice = append(slice, "a")
    slice[0] = "b"
    slice[1] = "b"
    fmt.Print(slice)
  }(slice)
  fmt.Print(slice)
}
```

Ответ:
```
Выводит: внутри анонимной функции - [b b a], снаружи в main - [a a]. Как в прошлом примере, после append() у нас создается новый slice,
после выхода из функции мы распечатываем оригинальный. Если сделать append() перенсти в конец, то получим [b b a], снаружи в main - [b b],
тк мы сначала изменим оригинал, а потом с помощью append расширим и сделаем копию с новым элементом.
```
