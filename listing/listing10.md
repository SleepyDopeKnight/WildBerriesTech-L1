### Что выведет данная программа и почему?
```
func update(p *int) {
  b := 2
  p = &b
}

func main() {
  var (
    a = 1
    p = &a
  )
  fmt.Println(*p)
  update(p)
  fmt.Println(*p)
}
```

Ответ:
```
Выводит: 1, 1. Это происходит потому что в функции update() передается копия и создается новый указатель на локальную переменную b.
Если мы сделаем *p = &b или будет передавать в сигнатуру **int, в функции update(), то будет вывод 1, 2.
```
